Featureban

Featureban is a simple Kanban simulation game. It has been used by trainers and coaches in Lean, Agile and Kanban related events the world over.

  • Iteration 1: The game starts simple with visual management
  • Iteration 2: Limits are introduced to the second and third column
  • Iteration 3: Some metrics are captured.

    • About Digital Featureban Game

    Digital Featureban Game is a tool that contains the same elements necessary to implement the game of Featureban. Its objective is to reduce the costs of the material, as well as to avoid its waste. It allows players to make mistakes so that they can learn from them.
